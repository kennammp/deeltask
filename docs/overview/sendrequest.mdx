## Requirements

To make a request you will need:
- `access_token`
- `Subscription_key`
- `companyId` or `employeeId` (depending on the request)

If you have completed the tasks in the [**Authorization**](auth) section, you should already have your `access_token` and `subscription_key`. You will include them as headers in your requests in the following format:

```bash
Authorization: "Bearer " + {{access_token}}
X-Subscription-Key: {{subscription_key}} 
```

If you do not know the `companyId` or `employeeId`, you will use the following Get requests to list them:
